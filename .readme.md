# 📈 Simple Linear Regression - Height vs Weight Prediction

This project demonstrates a **Simple Linear Regression model** built using **Python and Scikit-learn** to predict **Height based on Weight**.

It is a beginner-friendly **Supervised Machine Learning regression project** that covers the complete ML workflow from data preprocessing to model evaluation.

---

## 🚀 Project Workflow

The project follows these machine learning steps:

1. **Load Dataset**
2. **Divide features into independent and dependent variables**
3. **Train-Test Split**
4. **Feature Standardization**
5. **Train the Linear Regression model**
6. **Make predictions**
7. **Evaluate model performance**
8. **Predict on new custom data**
9. **Check regression assumptions using residuals**

---

## 📂 Dataset

The dataset contains:

- **Weight → Independent Feature**
- **Height → Dependent Feature**

### Sample Data

| Weight | Height |
| ------ | ------ |
| 45     | 120    |
| 58     | 135    |
| 48     | 123    |

## 📦 Installation

Clone the repository:

```bash
git clone <your-repo-url>
cd Supervised-learning

```

Create virtual environment:

python -m venv .venv

Activate virtual environment:

Windows
.\.venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt
▶️ Run the Notebook

Open Jupyter notebook in VS Code or run:

jupyter notebook

Open the notebook file and execute all cells.
